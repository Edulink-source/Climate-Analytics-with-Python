# Climate Change Data Analysis and Visualization

## Project Overview

This project explores the relationship between global temperature anomalies and CO₂ emissions per capita using Python-based data analysis and visualization techniques.

The analysis includes data cleaning, preprocessing, exploratory data analysis (EDA), statistical correlation analysis, and visualizations that highlight climate trends from 1950 onwards.

## Objectives

- Analyze historical global temperature anomalies.
- Analyze global CO₂ emissions per capita.
- Investigate the correlation between temperature changes and CO₂ emissions.
- Visualize long-term climate trends using informative charts.
- Compare CO₂ emissions across selected countries.

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Sources

The project uses publicly available climate datasets imported from Google Sheets:

- Global Temperature Data
- Global CO₂ Emissions Data

## Project Workflow

### 1. Data Collection
- Load temperature and CO₂ datasets.

### 2. Data Cleaning
- Handle missing values.
- Remove duplicate records.
- Filter data from 1950 onwards.

### 3. Data Preparation
- Select relevant variables.
- Rename columns for consistency.
- Merge datasets using the year field.

### 4. Exploratory Data Analysis
- Summary statistics.
- Distribution analysis.
- Trend analysis over time.

### 5. Visualization
- Global temperature anomaly trends.
- CO₂ emissions per capita trends.
- Correlation heatmap.
- Regression analysis.
- Country-level CO₂ comparison.

### 6. Correlation Analysis
- Calculate Pearson correlation between:
  - Temperature anomalies
  - CO₂ emissions per capita

## Key Findings

- Global temperatures have generally increased over time.
- CO₂ emissions per capita have shown significant growth since 1950.
- A strong positive correlation exists between CO₂ emissions and global temperature anomalies.
- Country-level analysis reveals substantial differences in emission patterns.
